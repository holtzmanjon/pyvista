Welcome to pyvista's documentation!
===================================

pyvista is package that provides image display, processing, and analysis tools for 
a versatile quick analysis environment

Dependencies:
   numpy
   scipy
   astropy
   bz2
   autopy (optional but valuable, for keystroke controlled moves in the display)
